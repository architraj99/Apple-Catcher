# Apple Catcher

I made this simple yet fun browser game where you catch falling apples in a basket and try to get the highest score possible.

## Features

* Catch falling apples in the basket using mouse, keys or touch controls
* Red apples give single point while golden ones are worth 5 each(Grrreeedy)
* Live Score tracking
* Lives system(Total 3)
* Best score is saved in local storage so score high n show off to ur friends anytime bleh
* Pause with the Space key
* Restart with the Escape key
* Responsive layout for desktop, tablet, and mobile

## How to Run

* Click on the link
* Simply click on the Start Game button
* Catch 'em apples!

## Tech Used

* HTML
* CSS
* javaScript
